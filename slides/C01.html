How does a computer work?

A computer takes input, processes it, and gives output.

Computer history

  • 1100 BCE: Abacus, analog computer
  • 1623: The first mechanical computer
  • 1950s: The first transistor computers
  • 1960s: The first integrated circuit computers
  • 1970s: The first microprocessor computers

Early computers

Gears and levers, no electricity

Software

  • Programming is the process of writing instructions for a computer to follow.
  • A program is a set of instructions for a computer to follow.
  • Code is the instructions that make up a program.

TL;DR

  • Early computers could only chang behavior by changing hardware
  • Modern computers can also change behavior by changing software

Instructing the CPU

  • The CPU works with binary instructions (x86, x64, ARM, Apple Silicon, etc.)
  • A common low level language for writing instructions is Assembly (`JMP`, `MOV`, `ADD`, etc.)
  • Various higher level language (C#, VB.Net, C++, JavaScript, TypeScript, etc.)
  • Human language?
Compare instructing a CPU with providing a human with instructions. A CPU is "dumb" compared to a human and needs really specific (as in low context) instructions, while a human is "smart" and can work with incomplete or non-specific instructions.

What happens when you press the power button?

Step by step

  1. CPU executes a hardcoded `JUMP` instruction to the first instructions on a ROM
  2. CPU starts executing BIOS directly from ROM
  3. BIOS loads bootloader from disk into RAM
  4. Bootloader loads OS kernel into RAM and hands over control

The BIOS is the very first application that is always run on the machine.

Terms

  • BIOS: Basic Input/Output System
  • CPU: Central Processing Unit
  • ROM: Read-Only Memory
  • RAM: Random Access Memory
  • OS: Operating System

.NET

Hello world... The hard way

  1. Write in notepad.exe: 
    System.Console.WriteLine("Hello world!");
  2. Save the file as HelloWorld.cs
  3. Open "Developer Command Prompt"
  4. Navigate to the folder where you saved the file (using cd to change directory)
  5. Type csc HelloWorld.cs
  6. Type HelloWorld.exe
